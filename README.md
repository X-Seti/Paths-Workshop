# Paths Workshop

GTA SA/VC/III path node editor with radar map overlay.

Base: [RadarWorkshop](https://github.com/X-Seti/Img-Factory-1.6) — radar tile viewer.
Part of the IMG Factory 1.6 modding suite.

## Planned Features
- Radar map display (SA 1,296 tiles)
- Path node overlay (vehicle + ped nodes)
- Click to select/edit nodes
- Link visualisation
- Export to binary nodes.dat

## Requirements
```
pip install PyQt6
```

## Usage
```bash
./paths_workshop.py
```
